# smechos-site

The SmechOS marketing site -- hand-authored vanilla HTML5/CSS3/JS, no
frameworks or build tools, hosted on GitHub Pages.

Originally built as a close-to-1:1 replica of an earlier Wix-hosted version
of this same site (see [Smech-Labs/SmechDeploy](https://github.com/Smech-Labs/SmechDeploy)'s
project history for that process), then rewritten with copy that actually
reflects SmechOS -- an independent Linux distribution with its own build
system, the `spk` package manager, and KDE Plasma 6 -- rather than generic
template text.

Build tooling and source for SmechOS itself live in
[Smech-Labs/SmechDeploy](https://github.com/Smech-Labs/SmechDeploy).
